## Getting Started

After cloning,

### FVM setup

it runs on flutter version: flutter: ">=3.18.0-18.0.pre.54", dart: ">=3.4.1 <4.0.0",

Install `fvm` if not already installed.

```bash
dart pub global activate fvm
```

Install the local `flutter` version.

```bash
fvm install
```
